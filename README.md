# studyfree-platform

StudyFree is a platform that helps students to find the best resources to study for their exams. It is a platform that is built by students for students. It is
free and open-source.

## Tech-Stack

**Frontend**

- NextJS
- TypeScript
- TailwindCSS
- Shadcn

**Backend**

- Go
- PostgreSQL
- SQLC
- MeiliSearch
